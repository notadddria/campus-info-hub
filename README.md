Ce este o resura (resource) in aplicatia ta?
- O resursă este un element extern folosit de aplicatie ca să funcționeze (ex: /images/facebook.png).

Da exemplu de un URI si explica componentele acestuia.
- index.html - contine declarație, metadate, resurse, conținut vizibil, etc.

Care parti sunt statice si care sunt dinamice?
- Datele statice nu se schimbă la rularea site-ului (Din CSS, HTML).
- Datele dinamice sunt încarcate la rularea site-ului (din JSON).

Este aplicatia ta document-centric sau interactive (sau ambele)? De ce?
- Aplicația mea este atât un document centric cât și interactivă.
- Pe pagina principală, înafară de textul HTML-ului și stil, este un filtru de căutare cu care se poate interacționa, deci e o pagină interactivă.
- Pe altă parte, celelalte pagii conțin doar text și stil, care o face o pagină document centric.
